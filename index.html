Welcome to My Website

Hi, this is He Junxian, a pasionate test engineer who love to explore and learn new things.

I am a ISTQB tester with manual testing and automated testing experience in different project and technological environments. I design test plan, write test cases, develop automated framework and test scripts by Python and execute test cases on different applications (web, app, embedded system) in the past 4 years.

I have experience with different flavors of Testing like Function, Automation, Exploratory Testing, UI and API Testing. I am good at using tools or frameworks like selenium, postman, wireshark, charlse, httprunner, QT4A, Git, etc. I am also familiar with agile and scrum.

I consider myself a fast learner and I prefer to knowing things well before I use or test it.
